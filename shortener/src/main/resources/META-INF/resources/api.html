API Documentation

Get familiar with the API to utilize link shortening for your projects.

POST

/b2b/short

  • data
    1. original - Original url to be redirected to
    2. shortened - Empty, will return the shortened link
    3. custom - Custom link to be used if available
    4. expiry - Expiration date for the link, in the DateTime format
  • dataType: JSON

GET

/b2b/analytics/{link}

  • parameters
    1. link - Shortened link to get the analytics for

GET

/{link}

  • parameters
    1. link - Shortened link to be redirected to the original
